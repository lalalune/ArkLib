  are permutations. Equivalently `E_r(μ_p) = Σ_{size-r multisets M}(orderings M)²` (the pure

  ⟹ `c=m(1,…,1)`; `Σc_i = r−r = 0 = mp ⟹ m=0 ⟹` a,b the same multiset. ∎

## Theme A (HEADLINE) — the complete perfect-B_r characterization of μ_n (subsumes A5)

Probe: `scripts/probes/probe_conj_Br_characterization.py`. Verified 20/20 cases, n∈{9,15,21,25,27}, r=2..5.

- **A6 (COMPLETE B_r characterization — PROVED).** For `n>1`, `μ_n ⊂ ℂ` is a perfect `B_r` set
  (the only solutions to `a_1+…+a_r = b_1+…+b_r` in `μ_n` are permutations) **iff**:
    · `n` even        →  `r = 1` only;
    · `n` odd prime   →  ALL `r`;
    · `n` odd composite, `q = smallest prime factor` →  `r < q`.
  **Proof (cyclotomic relation lattice / Lam–Leung):** a non-permutation solution gives a nonzero
  relation vector `c_v = mult_a(v)−mult_b(v)` with `Σc_v=0` (equal sizes) and `Σ_v c_v ζ^v = 0`. The
  relation lattice of n-th roots is generated by the μ_ℓ-coset sums (length ℓ) for primes `ℓ∣n`.
  · even: the length-2 antipodal `ζ^v+ζ^{v+n/2}=0` realizes a non-perm B_2 ⇒ fails for r≥2.
  · odd: shortest relation is a μ_q-coset sum (length q); an `r+/r−` pattern embeds it (a uses coset
    C₁, b a different coset C₂, both summing 0) iff `r ≥ q` — EXCEPT n prime, where there is only ONE
    coset (the full set) so a,b must be the same multiset (the size constraint `Σc_v=0=mp` forces the
    multiplier `m=0`). ∎
  **Decisive refutation-survival:** μ_25 perfect through B_4, first fails at B_5 (=minFac); μ_9/15/21/27
  first fail at B_3. 20/20 predictions matched — no counterexample.
  Subsumes A5 (prime case) and A1-even (the `r=1`-only / antipodal case). The cleanest statement of
  WHY 2-power (NTT) is hardest (fails B_2, maximal energy) and odd-prime is most rigid (perfect all r).
