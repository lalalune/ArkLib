## The chain of equalities (all in-tree / proven)

The prize floor `M = max_{b≠0}|η_b| ≤ √(2n log m)` follows (moment method, optimized at `r ≍ log m`) from
the **char-`p` sub-Gaussian moment bound**
```
E_r(μ_n, F_p)  ≤  (2r−1)!! · n^r          for r ≍ log m,   n = 2^μ,   p ≡ 1 (mod n).
```
Over ℂ this is the Lam–Leung theorem (proven, all `r`: vanishing sums of `2^μ`-th roots of unity are
generated by antipodal pairs `ζ^a + ζ^{a+n/2} = 0`, so the only `2r`-tuple coincidences are the
`(2r−1)!!` perfect matchings). Write the **char-`p` excess**
```
W_r := E_r(μ_n, F_p) − E_r(μ_n, ℂ)  ≥ 0.
```
`W_r = 0  ⟺  char-`p` faithful ⟹ floor holds. So the entire open core is: **bound `W_r` at depth `r≍log m`.**

## `W_r` is a count of short root-of-unity sums in a prime ideal

A char-`p`-only coincidence is a pair of `2r`-tuples with `Σ ζ^{a_i} ≡ Σ ζ^{b_j} (mod p)` but not over ℂ,
i.e. a **nonzero** `D = Σ_i ζ^{a_i} − Σ_j ζ^{b_j}` (a `±`-sum of `≤ 2r` roots of unity) lying in the
**prime ideal `P = ker(ℤ[ζ_n] → F_p, ζ ↦ h)`**, `h` a primitive `n`-th root of unity in `F_p`. So
```
W_r  =  #{ nonzero ±-sums of ≤ 2r of the n-th roots of unity that lie in P }.
```

## Why the easy proof fails — and where the threshold is

The tempting closure: `D` is a sum of `≤2r` roots of unity ⟹ `house(D) ≤ 2r`; if `D ∈ pℤ[ζ]\{0}` then
`house(D) ≥ p`, so `p > 2r ⟹ D = 0`, no excess. **This is wrong:** the condition is `D ∈ P`, not
`D ∈ pℤ[ζ]`. Because `p` splits completely (`F_p ⊇ μ_n`), `P` has **residue degree 1**, `N(P) = p`, so by
Minkowski its **minimum house is `~ p^{1/φ(n)} = p^{2/n}`, not `p`**. Hence:

> **The norm bound proves `W_r = 0` only while `2r < p^{2/n}`, i.e. `n < 2 log p / log(2r) ≈ 2 log p / loglog p`.**

This is the *small-subgroup* regime. In the **prize regime `n ≈ p^{1/4}`**, `p^{2/n} → 1` (numerically:
`p^{2/n} = 1.19` at `n=256`, `1.016` at `n=4096`, `1.0000` at the prize point `p=2^128, n=2^32`), so the
norm bound is **completely vacuous** — it forbids nothing. `P` has short vectors of house `≈ 1 ≪ 2r`, so
char-`p` coincidences are *a priori* possible. (`probe_geometry_of_numbers_threshold.py`.)

## The open core, named: "Mann's theorem mod P"

So `W_r` is governed by the *actual* number of short `±`-root-of-unity sums in `P`, and there is a sharp
tension:
- **Volume heuristic (random lattice):** `P` is a rank-`φ(n)` lattice of covolume `~ p·√disc`; the number
  of `±`-sums of `≤2r` roots of unity it contains is `~ (2r)^{φ(n)} / p`, which is `≫ 1` once
  `n > 2 log p / log(2r)` — predicting `W_r` *large* (floor false).
- **Measured reality (full enumeration in-regime, `probe_inregime_moment_split.py`):** `E_r/Wick ≤ 1` and
  *decreasing* in `r` — `W_r ≈ 0`, floor holds with margin.

The roots-of-unity arithmetic **suppresses** the count far below the volume heuristic. Over ℂ that
suppression is *exactly* Mann's theorem (minimal vanishing sums of `2^μ`-th roots = antipodal pairs only).
The open core is the mod-`P` analogue:

> **Prize core ⟺ a "Mann's theorem mod `P`":** the only short (`length ≤ 2r ≍ log p`) `±`-sums of `2^μ`-th
> roots of unity lying in `P` are the trivial ones (those already `= 0` over ℂ, i.e. antipodal-generated) —
> equivalently `P` contains no *short, genuinely-`p`-adic* vanishing relation among `2^μ`-th roots of unity
> of length `≤ 2 log p`.

This is a **closed, classical statement** (no coding theory, no analysis): bound the length of the shortest
nonzero element of `P` that is a `±`-sum of `n`-th roots of unity, for `n = 2^μ`, `p ≡ 1 (mod n)`, `n ≈ p^{1/4}`.
It is equivalent to BCHKS Conjecture 1.12 and to the prize floor. It is **open** — Mann/Lam–Leung control
the archimedean (`= 0`) relations completely, but there is no known theorem controlling the `p`-adic
(`∈ P`, `≠ 0`) short relations once `p^{2/n} → 1`. (A "Mann's theorem mod `P`" would close the prize; none
exists.)

## Why this is the right frontier (and what it rules out)

- It is **not** an analytic `√`-cancellation statement (so the BGK/di-Benedetto `n^{0.989}` machinery, which
  bounds the sum directly, is the wrong tool — it can't see the exact suppression).
- It is **not** second-order (so the meta-theorem's `√p` cap does not apply to *this* statement — the
  meta-theorem caps methods that use only `Σ η²`; the short-vector count is a genuinely higher object).
- It is a **lattice/Diophantine** question about a single explicit ideal `P` and the explicit finite set of
  `±`-root-of-unity sums — fully discrete, no asymptotic averaging.

The honest residue: proving the suppression (that the roots-of-unity structure keeps `P` free of short
relations) is the 25-year-open content. This note does not prove it; it states it as cleanly and classically
as the campaign has managed, and identifies the one missing theorem ("Mann mod `P`") by name.
